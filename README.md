# 📖 About The Project

Safar Sathi CG is a Smart Eco & Cultural Tourism Platform .

The project aims to digitally promote tourism in Chhattisgarh by combining:

✅ Smart Tourism  
✅ Eco Tourism  
✅ Tribal Culture Promotion  
✅ Tourist Safety  
✅ Trip Planning  
✅ Local Handicraft Marketplace  

The platform helps tourists discover beautiful destinations, plan trips, check safety information, explore tribal culture, and interact with an AI-style tourism assistant.

---

# 🌍 Project Vision

Many beautiful tourist destinations in Chhattisgarh remain unexplored due to lack of awareness and digital accessibility.

Safar Sathi CG provides a centralized tourism platform where users can:

- Explore tourist destinations
- View detailed place information
- Plan trips easily
- Check nearby hospitals & hotels
- Explore tribal handicrafts
- Discover ongoing cultural festivals
- Use an AI-style tourism chatbot

# 🚀 Installation & Setup

## 1️⃣ Clone Repository

```bash
git clone https://github.com/your-username/safar-sathi-cg.git
```

---

## 2️⃣ Navigate to Project Folder

```bash
cd safar-sathi-cg
```

---

## 3️⃣ Install Dependencies

```bash
npm install
```

---

## 4️⃣ Start Development Server

```bash
npm run dev
```
